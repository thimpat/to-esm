## Description !heading

A tool to convert Commonjs files into ESM.

<br>

---

## Installation !heading

```shell

npm install to-esm -g

```

<br>

---


## Usage !heading



```shell

to-esm <filepath> [--output=<dirpath>] [--html=<filepath>] [--noheader] [--target=< browser|esm >] 
[--bundle=<filepath>] [--update-all]

```
